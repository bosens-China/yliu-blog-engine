# 核心特性

本项目旨在提供一个功能完备且体验一流的现代化博客解决方案。

### 现代化与高性能

- **技术栈**: 基于业界前沿的 Next.js 15, React 19, TypeScript, และ Tailwind CSS v4，保证了项目的可维护性和未来的可扩展性。
- **性能**: 默认采用静态站点生成 (SSG)，所有页面在构建时预渲染为 HTML，为访问者提供极致的加载速度，并对 SEO 极其友好。
- **响应式设计**: 完美适配桌面、平板和移动设备。我们特别为 iOS 设备的刘海屏和动态岛优化了安全区域，确保在所有设备上都有一致的、沉浸式的阅读体验。

### 以 GitHub 为中心的内容流

- **Issues 即文章**: 你只需要在 GitHub 仓库中创建 Issue，它的标题、内容和标签 (Labels) 就会自动成为你博客的文章、正文和分类。
- **零感发布**: 无需登录任何后台。文章的发布、更新、隐藏（通过 close issue）都直接在 GitHub 完成，自动触发更新。
- **关于页面**: 在仓库根目录放置一个 `about.md` 文件，它会自动成为你的“关于我”页面，保持了内容来源的一致性。

### 强大的内置功能

- **动态导航**: 博客的导航菜单（如分类、专栏、关于我）会根据你的实际内容自动显示或隐藏。如果没有专栏文章，"专栏"菜单就不会出现。
- **模糊搜索**: 内置了基于 `fuse.js` 的强大前端模糊搜索功能，支持按标题、摘要、标签等多个维度进行加权搜索。
- **精致主题**: 提供了精心设计的亮色与深色主题。深色模式参考了 Cursor 编辑器的设计，采用渐变背景和柔和的卡片样式，提供舒适的夜间阅读体验。

### AI 驱动（可选）

- **智能 SEO**: 可对接 AI 服务，自动为每篇文章生成更吸引人的 SEO 标题、描述和关键词。
- **专栏分析**: AI 可以智能分析你所有文章的标题，自动将相关的文章组织成“专栏”或“系列”，无需手动配置。
- **优雅降级**: 所有 AI 功能都是可选的。即使不配置任何 AI 服务，系统也会回退到基于规则的基础算法，确保核心功能正常可用。

### 高度可定制与可扩展

- **环境变量配置**: 从博客标题、作者名到页脚文本，都可以通过环境变量轻松定制。
- **自定义菜单**: `Header` 导航栏的菜单项可以通过一个简单的 JSON 配置进行完全的自定义。
- **插件化脚本**: 底层的数据处理脚本采用插件化架构，为深度开发者提供了极高的可扩展性，方便添加新的数据源或处理逻辑。
- **GitHub Action 驱动**: 内置了强大的 GitHub Action，不仅可以自动部署，还支持定时任务、条件部署等高级 CI/CD 工作流。
