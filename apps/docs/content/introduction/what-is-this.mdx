# 项目是什么？

**YLiu Blog Engine** 是一个基于 Next.js 15 和 React 19 构建的现代化、高性能、易于部署的博客系统。

它的核心理念是 **“内容与代码分离，专注写作本身”**。

我们相信，对于开发者而言，最自然、最无干扰的写作平台就是他们每天都在使用的工具——代码编辑器和 GitHub。因此，本项目的核心设计就是将你的 **GitHub Issues** 无缝转化为一篇篇排版优雅、功能丰富的博客文章。

### 这个项目解决了什么问题？

传统的博客系统往往面临以下一个或多个痛点：

- **繁琐的后台管理**: 需要登录独立的后台系统，使用不熟悉的编辑器，操作繁琐。
- **复杂的部署维护**: 需要自己购买服务器、配置数据库、处理安全更新。
- **技术栈陈旧**: 很多静态站点生成器虽然经典，但在技术栈和开发体验上可能已经落后。
- **定制困难**: 想修改主题或增加功能，往往需要深入了解一个庞大的系统。

**YLiu Blog Engine** 旨在通过以下方式解决这些问题：

- **以 GitHub 为中心**: 你只需要在自己的仓库中创建和管理 Issues，其他的一切都由自动化流程完成。
- **零数据库，零后台**: 整个系统是静态的，除了 GitHub API 外没有任何外部依赖。
- **现代化技术栈**: 享受 Next.js 带来的极致性能、服务端组件、以及丰富的 React 生态。
- **为开发者设计**: 提供清晰的配置选项和插件化的脚本系统，让深度定制变得简单。

### 谁适合使用这个项目？

- **希望快速拥有个人博客的开发者**: 你可以在 5 分钟内，通过我们的 GitHub Action 模板一键部署属于自己的博客。
- **寻求现代化博客方案的创作者**: 如果你对性能、SEO、和现代 Web 技术有要求，这套系统提供了一个极佳的基础。
- **希望学习 Monorepo 和 Next.js 实践的开发者**: 本项目采用 pnpm + Turborepo（虽然目前简化了）的 Monorepo 结构，包含了数据处理、前端渲染、类型共享等多个工程实践，是一个很好的学习案例。
